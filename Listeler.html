Sıralı Listeler

Sıralı listeler oluşturmak için html ol (ordered list) etiketini kullanıyoruz. Listenin herbir elemanını li etiketi ile ve listeleme sembolünü (numaralar, harfler, romen rakamları gib) de type özelliği ile belirliyoruz. Sıralamanın kaçıncı değerden başlayacağını da istersek Start özelliği ile değiştirebiliyoruz.

Eğitimlerimiz

  1. Yazılım Eğitimleri
    1. C#.Net Core
    2. Asp.Net Core
    3. Entity Framework Core
  2. Veritabanı Eöitimleri
    1. MS-Sql Server
    2. My-Sql
    3. Oracle Server
  3. Mobile Eğitimleri

Sırasız Listeler

Sırasız listeler oluşturmak için html ul (unordered list) etiketini kullanıyoruz. Listenin herbir elemanını li (list item) etiketi ile ve listeleme sembolünü (disc, square, circle gibi) de type özelliği ile belirliyoruz. Sıralamanın kaçıncı değerden başlayacağını da istersek Start özelliği ile değiştirebiliyoruz.

  • Yazılım Eğitimleri
    • C#.Net Core
    • Asp.Net Core

  • Veritabanı Eöitimleri

    • My-Sql
    • Oracle Server
      • PL-Sql
  • Mobile Eğitimleri

    • Flutter
